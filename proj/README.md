# mini-projets
## Organisation pratique
* Le travail s'effectue par binôme.
* chaque binôme doit produire
  * un **code** python par binôme (évalué sur 12 points)
  * un **document** d'explication par élève (évalué sur 8 points), au format numérique (fichier texte, présentation...)
#### travail à rendre

* le **code rendu** doit inclure des exemples (servant de démo).
* le **document d'explication** doit présenter le code, en précisant notamment : 
  * la contribution de chaque élève au travail du groupe
  * la spécification de chaque fonction : 
    * le rôle et le type des paramètres
    * le rôle et le type de la valeur renvoyée
    * le traitement réalisé par la fonction
    * choix des structures d'exécution (boucle, conditionnelle, etc...)
    * les tests réalisés pour cette fonction
